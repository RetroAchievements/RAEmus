1)  RAQUASI88 とは？
-------------------------------------------------------------------------------

      RAQUASI88 は、 RetroAchievementsという実績システムに対応するビルドです。
    Win32版　QUASI88 を元にしたもので、Windowsでしかコンパイル不可能となりますが、
    バイナリは WINE 等で UNIX 上でも動かせるかもしれません。

      QUASI88　の一般情報には MANUAL.TXT と QUASI88.TXT を参照ください。

-------------------------------------------------------------------------------
2)  RetroAchievementsとは？
-------------------------------------------------------------------------------

      RetroAchievements は、 レトロゲーム向けの実績システムです。
    様々なプラットフォームでゲームをプレイしながら有志メンバーによって作られた実績を解除
    できるシステムです。ただし、言語サポートは英語のみです。

      詳しくはウェブサイト ( https://retroachievements.org/ )　を参照ください。

-------------------------------------------------------------------------------
3)  QUASI88との違い
-------------------------------------------------------------------------------

    1. パフォーマンス

    RetroAchievements の機能を実装する為に、パフォーマンスが少し下がりました。
    CPU クロックを高く上げたり、速度 400% か　-nowait をセットしたりする以外は、
    現在のパソコンの性能で充分動かせると思われます。

    2. 機能制限

    不正行為を防ぐために、RetroAchievements でログインして Hardcore Mode を
    セットすると、幾つかの機能が制限されます。
        - イメージを一度セットすると、実績対応の別のタイトルに切り替えるには
          ディスク・テープを取り外して、システムをリセットすることが必要となります。
        - メニューバーで速度を 100% 未満にセットすることができなくなります。
        - ステートロードができなくなります。
    なお、メニューシステムの幾つかの機能が常に制限されています。
        - 速度を 100% 未満にセットすることができなくなります。
        - CPU クロックを 4MHz 未満にセットすることができなくなります。

    3. 新機能

    RetroAchievementsの機能をすべて実装。
        - ログインして実績対応のゲーム(ディスク・テープ)イメージを挿入すると、
          実績を解除することができます。
          ※　実績対応のゲームは、イメージファイルのコンテンツで検出されているため、
            ディスクイメージの上にセーブしたり、プロテクトを変わったりすることで、
            検出不可能になります。詳しくは使い方の項目を参照ください。
          ※ 実績対応のイメージファイルは大概、 RetroAchievements のウェブサイトでの
            対応ゲームページからアクセスできる掲示板に記載されています。
        - 起動中の対応ゲームの情報、ログイン中のユーザー情報、実績の情報等にアクセスする
          ことができます。
        - 実績の不具合(解除されない実績、違うところで解除される実績等)を報告する
          ことができます。

        詳しくは RetroAchievements のウェブサイトを参照ください。

-------------------------------------------------------------------------------
4)  コンパイル・インストール方法
-------------------------------------------------------------------------------

    1. git submodule update --init で RAIntegration のサブモジュールを初期化する。
    2. BUILD-WIN32.TXT を参照して、 RAQUASI88.sln をコンパイルする(SDL不要)。

-------------------------------------------------------------------------------
5)  使い方
-------------------------------------------------------------------------------

    1. アカウント成作

      実績を解除するには RetroAchievements のウェブサイトでアカウントを作成することが必要となります。
      「create a new account」 をクリックして、ユーザー名・メールアドレス・パスワードを入力することで完了。
      ユーザー情報(個人メッセージ・アイコン等)をセットすることができます。

    2. RAQUASI88 でログイン

      初めて起動するときは RAIntegration.dll の更新が必要となります。ダウンロード完了したら
      ログインすることができます。ユーザー名とパスワードを入力してログイン完了。

    3. イメージファイルを挿入
    
      普通にイメージファイルを挿入したら、実績対応のゲームが検出されたらメッセージが表示されます。
      ディスクイメージはセーブ等でファイルコンテンツが変わると検出不可能になるため、 SAVE フォルダで
      セーブ用のコピーが作成され、選択するイメージを上書きせずにセーブすることができます。

      また、Hardcore Mode をセットすると、別のタイトルを挿入するには、イメージの取り外しとリセットが必要です。
